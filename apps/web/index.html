Figma to Elementor platform

Turn Figma frames into importable Elementor templates.

This MVP ships the product spine: landing experience, local conversion API, Figma plugin shell, and a first-pass mapper that generates Elementor JSON from a simplified Figma tree.

Pipeline

  1. Create or copy an API key
  2. Install the Figma plugin
  3. Select a Figma frame
  4. Convert and import into WordPress
