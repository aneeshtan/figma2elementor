# Figma Plugin Listing Copy

## Name

Figma2Element Converter

## Tagline

Convert Figma frames into Elementor template JSON.

## Short description

Figma2Element Converter takes the selected Figma frame, sends its structure to the Figma2Element API, and returns Elementor-ready JSON that can be downloaded and imported into WordPress.

## Full description

Figma2Element Converter helps designers and builders move faster from design to page-builder output.

Use the plugin to:

- inspect the current Figma selection
- send selected frames to the Figma2Element conversion API
- generate Elementor-compatible template JSON
- download the export and import it into Elementor

Current capabilities:

- auto-layout frame to container mapping
- text to heading and text widgets
- simple button detection
- image placeholder mapping
- authenticated exports with API keys

Best for:

- Figma to Elementor workflows
- rapid landing-page prototyping
- early-stage design-to-build pipelines
